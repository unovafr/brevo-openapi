# Brevo::GetSsoToken

## Properties

| Name | Type | Description | Notes |
| ---- | ---- | ----------- | ----- |
| **token** | **String** | Session token. It will remain valid for a short period of time only. |  |

## Example

```ruby
require 'brevo'

instance = Brevo::GetSsoToken.new(
  token: ede520dxxxxxxxxxxxx76d631fba2
)
```
